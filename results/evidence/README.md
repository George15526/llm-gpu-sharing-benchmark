# Experiment Evidence

This directory contains curated benchmark evidence for comparing NVIDIA GPU time-slicing and HAMi GPU sharing.

## Contents

- `summary.csv`: Aggregated benchmark metrics.
- `timeslicing/`: NVIDIA time-slicing benchmark summaries.
- `hami/`: HAMi benchmark summaries.
- `figures/`: Generated charts for throughput, latency, and GPU utilization.

Raw logs and temporary outputs are intentionally excluded from this repository.
